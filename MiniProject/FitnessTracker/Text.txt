🔹 Project Features

Workouts

Add/Edit/Delete workouts.

Each workout has: name, type (cardio/strength/yoga), duration, calories burned, date.
Track Progress
Show weekly/monthly progress (with chart.js or recharts).
Display stats (total workouts, total calories, streaks).
Goals
User can set fitness goals (e.g., run 10 times this month).
Progress bar towards goal.
Calendar
Show all workouts on a calendar (react-calendar or react-big-calendar).
Notifications (basic)
Simple “reminder” popup using setTimeout or Notifications API.
UI Extras
Dark/Light mode toggle.
Smooth animations (framer-motion).
